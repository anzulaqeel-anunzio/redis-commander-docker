# Redis Commander Docker Setup

A Docker Compose configuration to spin up Redis alongside Redis Commander (a web-based GUI for Redis).

## Features

-   **Redis Commander**: Web GUI to view/edit keys.
-   **Redis Server**: Latest stable Redis instance.
-   **No Install**: Completely containerized.

## Installation

1.  **Clone the repository**.
2.  **Start Services**:
    ```bash
    docker-compose up -d
    ```
3.  Visit `http://localhost:8081` to manage your Redis instance.

## Usage

-   **Redis Port**: 6379 (Internal) / Exposed if needed.
-   **GUI Port**: 8081.
